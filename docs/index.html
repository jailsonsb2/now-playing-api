Now Playing API

Esta página documenta a API hospedada em OCI. O objetivo é explicar o funcionamento do serviço, mostrar exemplos de uso e dar visibilidade ao trabalho, sem publicar o código executável.

Testar o endpoint Abrir exemplo

O que a API entrega

Metadados em tempo real

Receba o título, artista, álbum, capa e duração atual da música em execução.

Histórico de faixas

Monte cards de reprodução recente com os mesmos dados usados por players.

Clip mode

O campo youtubeId permite vincular o vídeo oficial da música ao player.

Endpoint público

GET https://api.twj.es/?url=https://stream.zeno.fm/yn65fsaurfhvv
