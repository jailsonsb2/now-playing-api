Now Playing API

Consuma metadados em tempo real para rádios web com capa, histórico, contexto visual e busca de faixas — tudo via endpoint público. Sem código da API publicado neste repositório.

Testar agora Ver exemplo

O que a API entrega

Now playing

Receba a música atual com artista, álbum, capa, duração e identificador de vídeo.

Histórico

Mostre as últimas faixas tocadas no player ou painel, com dados consistentes.

Busca de contexto

Busque capa e álbum por artista e título usando `search.php`.

Endpoints públicos

Metadata em tempo real

GET https://api.twj.es/?url=https://stream.zeno.fm/yn65fsaurfhvv

Retorna o estado da faixa atualmente em reprodução.

Busca de capa e contexto

GET https://api.twj.es/search.php?query=Ministério%20Vineyard%20-%20Tu%20És%20Bom

Use o parâmetro query com artista e título para obter capa, álbum e correspondências relevantes.

O retorno inclui results como correspondência principal e other_matches como alternativas. Campos como artwork e stream_url são URLs diretas.

Dashboard público para observabilidade: https://twj.es/dashboard
