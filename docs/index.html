Endpoints públicos

Metadata em tempo real

GET https://api.twj.es/?url=https://stream.zeno.fm/yn65fsaurfhvv

Retorna o estado da faixa atualmente em reprodução.

Busca de capa e contexto

GET https://api.twj.es/search.php?query=Ministério%20Vineyard%20-%20Tu%20És%20Bom

Use o parâmetro query com artista e título para obter capa, álbum e correspondências relevantes.

O retorno inclui results como correspondência principal e other_matches como alternativas. Campos como artwork e stream_url são URLs diretas.

Dashboard público para observabilidade: https://twj.es/dashboard

Public endpoints

Real-time metadata

GET https://api.twj.es/?url=https://stream.zeno.fm/yn65fsaurfhvv

Returns the current playing track state.

Cover and context search

GET https://api.twj.es/search.php?query=Ministério%20Vineyard%20-%20Tu%20És%20Bom

Use the query parameter with artist and title to get cover, album and relevant matches.

The response includes results as the main match and other_matches as alternatives. Fields such as artwork and stream_url are direct URLs.

Public observability dashboard: https://twj.es/dashboard

Endpoints públicos

Metadatos en tiempo real

GET https://api.twj.es/?url=https://stream.zeno.fm/yn65fsaurfhvv

Devuelve el estado de la pista que se está reproduciendo actualmente.

Búsqueda de portada y contexto

GET https://api.twj.es/search.php?query=Ministério%20Vineyard%20-%20Tu%20És%20Bom

Usa el parámetro query con artista y título para obtener portada, álbum y coincidencias relevantes.

La respuesta incluye results como coincidencia principal y other_matches como alternativas. Campos como artwork y stream_url son URLs directas.

Panel público de observabilidad: https://twj.es/dashboard
