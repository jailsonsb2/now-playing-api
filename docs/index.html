Now Playing API

Esta página documenta a API Now Playing para rádios web, voltada para quem precisa de metadados de música em tempo real com qualidade, confiabilidade e integração simples em players, painéis e aplicações web.

Testar o endpoint Abrir exemplo

O que a API entrega

Metadados em tempo real

Receba o título, artista, álbum, capa e duração atual da música em execução para enriquecer qualquer experiência de rádio.

Histórico de faixas

Monte cards de reprodução recente com os mesmos dados usados por players e painéis modernos.

Busca de capa e contexto

Use os endpoints de metadata e contexto para trazer capa, álbum e informações visuais de forma consistente.

Endpoints públicos

GET https://api.twj.es/?url=https://stream.zeno.fm/yn65fsaurfhvv

Endpoint principal de metadados para a faixa em reprodução.

GET https://api.twj.es/search?...

Endpoint de busca de contexto e capa para enriquecer a experiência visual do player.
